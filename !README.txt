By-time-003 - Android-приложение. Выбирает из базы данных SQLite произвольную строку. Строка является идеей для досуга. Таблицы в БД приведены к первой нормальной форме. Настройки не изменяются, созданы с целью изучить компонент ListView.
 
Quick_Sort - быстрая сортировка массива.

TestSwingComponent - использование GUI Swing.

AnnuitPayments - консольная программа, вычисляющая аннуитентные платежи по кредиту.

UsingExceptionsThreadsInheritanceCollections: 
 Один поток пишет данные в список с произвольным интервалом. Второй читает оттуда данные раз в секунду и выводит на экран. Если данных для чтения нет - бросает исключение и работа приложения завершается.
